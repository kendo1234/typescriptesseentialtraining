# Typescript Essential Training

A LinkedIn Learning course that covers a high level overview of Typescript 

## Installation

Node is required for this project

```bash
npm i
```

Globally install Typescript compiler 

```bash
npm i -g typescript
```

## Usage

Compile Typescript

```bash
tsc
```
